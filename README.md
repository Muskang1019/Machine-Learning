# Loan Approval Prediction using Machine Learning

## 📌 Project Overview
This project predicts whether a loan application will be approved using various Machine Learning classification algorithms. The project includes data preprocessing, model training, evaluation, and performance comparison.

## 📂 Project Structure

```
├── DT_LAD.ipynb                 # Decision Tree Model
├── knn.ipynb                    # K-Nearest Neighbors Model
├── RandomF(1).ipynb             # Random Forest Model
├── xgboost.ipynb                # XGBoost Model
├── loan_approval_dataset.csv.xls
├── loan_data.csv(1).xls
└── README.md
```
